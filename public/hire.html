Modern Language Association

October 2006 – present

As Head of Software Development, I lead our digital publishing platform team, automating workflows, modernizing processes, and crafting digital products. I also lead and code alongside our development team on the association’s Web sites and scholarly communication projects, and architect the systems they run on.

MLA Commons

commons.mla.org

Led development on an open-source scholarly communication platform, built on WordPress MultiSite and Commons-in-a-Box, that provides a social network and publishing platform for 6,000+ members. Introduced a modern WordPress architecture using Composer, Ansible, Packer, and AWS.

MLA Convention Mobile Program

mla16.org, github.com/mlaa/mla16.org

A single-page HTML5 app, built with Backbone Marionette, for the MLA’s annual convention, which holds 800+ sessions over four days. App used by over 6,000 unique users. Prerendered site using PhantomJS for search engine discovery.

REST API

Proposed, architected, documented, and led development of a secure RESTful Web API as a gateway between our new eZ Publish CMS and a legacy Oracle database. Supported its use by multiple software contractors. Conducted extensive data modeling to ensure that the API outlives the underlying database and can be used with future systems.

Literary Research Guide

mlalrg.org

Reconceived a print bibliography as an XML-first subscription-based digital publication with a robust interface, research tools, and hooks into library catalogs, Google Books, and WorldCat. Publication received Choice award for “best e-product in the humanities” and was recognized by Library Journal.

MLA Handbook

mlahandbook.org

First Web site for the award-winning MLA publication. Publication received Choice award for “Outstanding Academic Title.”

MLA Journals

mlajournals.org

Developed an automated workflow using Bash, InDesign JavaScript, XSLT, and other tools to accept InDesign documents, convert them to JATS XML and archival quality PDFs (plus Web-ready versions), and send them to our online publishing platform, all with minimal human QA.
